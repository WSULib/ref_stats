How to Use

Recording Reference Statistics

Select the reference desk location from where you will be recording reference statistics.   When selected, the Please Select Your Location button will show a drop down menu listing the appropriate libraries.

After selected, the screen will notify you that your reference location been set.

Now that you’ve set your location, you can begin using the Reference Stats Tool.

After a reference transaction has occurred, click the appropriate button to record the type of transaction.

After you click the button, the transaction will logged by RefStats, and a confirmation message will display at the top:

You can edit this transaction immediately (perhaps you made a mistake and clicked the wrong button).  To do this, you simply click the message; it will take you to the edit screen.  For more information on editing this transaction, please click here to navigate to the editing transactions section.

You can also see that the transaction was updated in Today’s Stats, displayed at the bottom of the screen.  The statistics are colored coded according to transaction type.  Gold represents directional transactions; green represents brief reference; finally, black represents an extended reference transaction.

Editing Transactions

To edit recorded reference transactions, click the “Edit Transactions” button just below the main recording buttons:

Figure: Edit Transactions button

On the next screen, you will be presented with a table showing reference statistics for your currently selected location from the main screen.  Note: the main tool is designed to be in a small, narrow window, this editing area is not.  It is recommended that you make the window bigger to more efficiently edit transactions.

Figure: Edit Transactions main screen

When you begin editing, your location will be set to your location from the main page.  You can also view / edit transactions from other locations by using the “Where would you like to edit” menu as shown here:

Figure: “Where would you like to edit” menu

Note: Changing your location from this menu is temporary - returning to the main screen will revert back to the your currently selected location, and creating new transactions will default to your currently selected location as well.  This is to discourage viewing stats from another location, forgetting you changed locations, then recording transactions for that location.  Selecting your location on the main screen is the preferred, and most enduring way to set your location.

From this screen you are able to create new transactions, edit and delete previously recorded transactions, cycle through different days, and at the bottom, see transactions for the currently selected day.  

To create a new transaction, click the “New Transaction” button at the top.  This will bring up the following window:

Figure: Add New Transactions window

From here you are able to add a new transaction, selecting the location, reference type, and hour window.

If you need to revise a previously recorded transaction, you can click “Edit” from the table, under the “Actions” column:

Figure: Click “Edit” to edit transaction

This will bring up the following “Edit Transaction” window:

Figure: Edit Single Transactions window

From this window, you are able to change the reference type, the location of the transactions, the IP of the computer where the transaction took place (not recommended to change), and select from an hour window for the transaction.  Note: when editing a transaction, you may only select the hour for which the transaction was recorded, not the minutes or seconds.  

If you need to delete a transaction, simply click “Delete”.  Note: Deleting a transaction from the transactions table is immediate, and irreversible.  But remember, you can add them again if need be!
